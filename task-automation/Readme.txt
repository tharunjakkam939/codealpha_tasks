# Task Automation Scripts ⚙️

A collection of scripts to automate simple tasks.

## Scripts

- **move_images.py**: Moves `.jpg` files from one folder to another.
- **extract_emails.py**: Extracts email addresses from a `.txt` file.
- **scrape_title.py**: Scrapes the title of a fixed webpage.

## Run Example
```bash
python move_images.py
python extract_emails.py
python scrape_title.py
```
